Wiki the word, and other labels for the basic idea..[edit]

The basic meaning of the word is lost, or has become near-lost, because of the world wide web use has become much less like that which it was in the late nineteen nineties, wherein web production was an involved process which required formatting "pages" in hypertext markup language HTML/format, using a text editor or code editor, and then deliver them to the web server machine by File Transfer FTP (using a remote file manager program). The essential meaning of "wiki" as derived is simply "fast document production" with some aspect of "collaboration" as per the elements and principles of the project. "Collaboration" being the main essence, not "quickness," "collab" or "colab" (as in co- "cooperative," and -lab "labor-ratio-ry"), suggested by myself (in.violatio.di.tyranos.)..
