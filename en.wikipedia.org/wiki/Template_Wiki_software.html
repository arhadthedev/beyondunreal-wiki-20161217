Usage[edit]

Place this template, {{wiki software}}, on each article about wiki software in order to allow readers to link to all articles.

Policy[edit]

Do not include red links to this template; please create a page and add relevant content first.

Do not include the links to articles about wiki software developers, notable websites using wiki software or other articles whose primary topic is not wiki software.
