The base class for all UT3 pickup factories.

Properties[edit]

Property group 'floating'[edit]

bRandomStart[edit]

Type: bool

For floating pickups (Health Vials and Health Packs), this pickup will start at a random height.

bDoVisibilityFadeIn[edit]
