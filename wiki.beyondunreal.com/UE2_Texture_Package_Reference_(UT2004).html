UE2:Texture Package Reference (UT2004)

From Unreal Wiki, The Unreal Engine Documentation Site
Jump to: navigation, search

This page lists the retail texture packages of UT2004 and ECE and gives a short description of the content of each package.

The content of packages can be reviewed without opening them in the editor by using the tool UTXgaffer.

Note: Do Not Alter Default Packages.

Generalities:

  • packs called "*HardwareBrush" are skins for static mesh packages of that name
  • packs from UT2004's Editor's Choice Edition are in this list marked with the suffix [ECE]
