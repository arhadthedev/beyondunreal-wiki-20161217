The class that writes the DM general stats

Properties[edit]

MultiKill[edit]

Type: array<name>

Grouping of stats for the category MULTIKILL

Default value, index 0: 'MULTIKILL_DOUBLEKILL'

Default value, index 1: 'MULTIKILL_MULTIKILL'

Default value, index 2: 'MULTIKILL_MEGAKILL'

Default value, index 3: 'MULTIKILL_ULTRAKILL'

Default value, index 4: 'MULTIKILL_MONSTERKILL'

Spree[edit]

Type: array<name>

Grouping of stats for the category SPREE

Default value, index 0: 'SPREE_KILLINGSPREE'

Default value, index 1: 'SPREE_RAMPAGE'

Default value, index 2: 'SPREE_DOMINATING'

Default value, index 3: 'SPREE_UNSTOPPABLE'

Default value, index 4: 'SPREE_GODLIKE'

Default value, index 5: 'SPREE_MASSACRE'

VehicleStatsClass[edit]

Type: class<UTLeaderboardWriteBase>

Class that holds the stats mapping/info for vehicles

Default value: Class'UTGame.UTLeaderboardWriteVehiclesDM'

VehicleWeaponsStatsClass[edit]

Type: class<UTLeaderboardWriteBase>

Class that holds the stats mapping/info for vehicle weapons

Default value: Class'UTGame.UTLeaderboardWriteVehicleWeaponsDM'

WeaponsStatsClass[edit]

Type: class<UTLeaderboardWriteBase>

Class that holds the stats mapping/info for weapons

Default value: Class'UTGame.UTLeaderboardWriteWeaponsDM'
