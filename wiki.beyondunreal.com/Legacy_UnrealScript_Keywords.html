Unreal Engine 3+ UnrealScript Keywords

Variables

  • [Databinding]
  • [DuplicateTransient]
  • [EditFixedSize]
  • [EditorOnly]
  • [Interp]
  • [NoClear]
  • [NoImport]
  • [Private{Private}]
  • [Protected{Protected}]
  • [Public{Private}]

Functions

  • [Client]
  • [Server]

Example:

reliable server function servermutate(string mutatestring)
  • [Virtual]
  • [Const]

Example:

native final function bool haslinkedops( optional bool bconsiderinputlinks ) const;
