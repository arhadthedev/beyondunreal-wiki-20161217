In [Date]* Epic Games released the UnrealEngine2 Runtime, a free version of the Unreal Engine 2.0 for non-commercial use.

Epic have also relased a non-free version of the Runtime that gives licenses access to the native code headers and some limited commercial distribution rights.

Can I make a game with the UnrealEngine2 Runtime?[edit]

No

When the Runtime was released the was a lot of confussion and arguments about what you could and couldn't do with the Runtime. The final version of the EULA now clearly states:

3. Commercial Exploitation. You may not use this Runtime Software, or any content created with it, or use the tools provided with this Runtime Software for any commercial purpose whatsoever.

4. Use Restrictions. <...> YOU MAY NOT USE THE RUNTIME SOFTWARE TO DEVELOP GAMES. YES, I KNOW THIS SOUNDS NASTY BUT LET’S FACE IT, EPIC‘S PRIMARY MEANS OF INCOME COMES FROM GAME SALES AND ENGINE LICENSING. WE WOULD BE SHOOTING OURSELVES IN THE FOOT IF WE ALLOWED AN ENGINE WE GAVE AWAY FOR FREE TO TRAMPLE OUR PRIMARY MEANS OF INCOME. SORRY, BUT IF YOU WANT TO DEVELOP A GAME PLEASE DEVELOP IT AS A MOD FOR ONE OF OUR EXISTING RETAIL GAME PRODUCTS OR CONTACT US REGARDING A PROPER ENGINE LICENSE.
