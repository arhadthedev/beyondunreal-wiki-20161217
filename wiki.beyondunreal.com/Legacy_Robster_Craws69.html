About Me[edit]

I enjoy UT. Been playing, modding, and mapping for it for quite a while (UT99 & UT2K4). I also run a 8-man VCTF test server for UT2K4.

My Website[edit]

I run a small website called the whoshack http://www.whoshack.com/. It has all of my downloads and a few other things as well there.

Notable Accomplishments - UT2K4[edit]

  • 2K4 NoSkilz Weapons
  • 2K4 Stupid Maps
  • 2K4 Skill Balancer
  • 2K4 Sentinels Mod (Didn't create it, just updated & fixed it)

Future Projects[edit]

I would like to make some ports of my existing maps and mods to UT3, plus I have some ideas I will work on if I have the time. I would also like to write some tutorials on UT3 mod development. Looking forwarding to creating some fun things with UT3.
